`brew update`

`brew install pyenv`

`pyenv install 3.7.12`

`run the command 'pyenv init --path', paste the output in your shell (~/.zshrc) at the end line, to enable shims.`\
`and after that add 'eval "$(pyenv init -)" '`\
`source ~/.zshrc`

`pyenv shell 3.7.12`
